# 백준 11501번 - 주식

[문제 보러 가기](https://www.acmicpc.net/problem/11501)

## 문제 요약
홍준이는 매일 3가지 중 한 행동을 한다.
1. 주식 하나를 산다.
2. 원하는 만큼 가지고 있는 주식을 판다.
3. 아무것도 안한다.

날 별로 주식의 가격을 알려주었을 때, 최대 이익이 얼마나 되는지 계산을 해야 한다.

## 입력
- 입력의 첫 줄에는 테스트케이스 수를 나타내는 자연수 T가 주어진다. 
- 각 테스트케이스 별로 첫 줄에는 날의 수를 나타내는 자연수 N(2 ≤ N ≤ 1,000,000)이 주어진다.
- 둘째 줄에는 날 별 주가를 나타내는 N개의 자연수들이 공백으로 구분되어 순서대로 주어진다. 
- 날 별 주가는 10,000이하다.


## 출력
- 각 테스트케이스 별로 최대 이익을 나타내는 정수 하나를 출력한다. 
- 답은 부호있는 64bit 정수형으로 표현 가능하다.

## 예제 입력 및 출력
링크의 예제 입력 및 출력 확인

## 문제풀이

`1.설계`

예시를 보면, 날 수가 3일이고, 날 별로 주가가 10, 7, 6일때 주가가 계속 감소하므로 최대 이익은 0이다.
다른 예시를 보면, 날 별로 3, 5, 9 일때, 처음 두 날에 주식을 하나씩 사고, 마지막 날 다 팔아 버리면 이익이 10이 된다.
날 별로 1, 1, 3, 1, 2인 경우 첫날, 둘쨌날, 넷쨌날에 주식을 사고, 3일째와 5일째에 팔면 최대 이익은 5이다.

오늘 날짜를 기준으로, 내일부터 마지막 날 까지 오늘보다 비싼 주식중 가장 비싼날에 다 팔 것이므로, 이 차를 계속 더해주면 된다.

`2-1.시간초과`

덱의 pop을 활용하려고 하니 시간초과가 발생 -> MAX 값을 지정하고 List에서 한번에 값을 계산

`2-2.시간초과`

MAX를 수행하면, 전체 리스트를 한번 훑게 되므로 시간이 오래 걸릴 수 밖에 없음.

MAX를 수행하지 않고, 최댓값을 확인하려면 어떻게 해야 하는가?

뒤에서 부터 확인 -> 현재 설정된 MAX값 보다 크면 새로운 MAX값으로 설정
