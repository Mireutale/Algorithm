# 백준 5567번 - 결혼식

[문제 보러 가기](https://www.acmicpc.net/problem/5567)

## 문제 요약
상근이는 결혼식에 동기 중 친구와 친구의 친구를 초대하기로 함.

동기는 모두 N명이고, 학번은 1부터 N이며 상근이의 학번은 1번임.

주어진 친구 관계 리스트를 통해서 결혼식에 초대할 사람의 수를 구하시오.

## 입력
- 첫째 줄에 상근이의 동기의 수 n (2 ≤ n ≤ 500)이 주어진다. 
- 둘째 줄에는 리스트의 길이 m (1 ≤ m ≤ 10000)이 주어진다. 
- 다음 줄부터 m개 줄에는 친구 관계 ai bi가 주어진다. (1 ≤ ai < bi ≤ n) ai와 bi가 친구라는 뜻이며, bi와 ai도 친구관계이다. 

## 출력
첫째 줄에 상근이의 결혼식에 초대하는 동기의 수를 출력한다.

## 예제 입력 및 출력
링크의 예제 입력 및 출력 확인

## 문제풀이
`1. 설계`

양방향으로 연결된 친구 관계 그래프를 생성하고, 상근이의 친구 & 친구의 친구까지를 모두 구하면 된다.

`2. 실패`

2606 바이러스 문제처럼, 풀고자 했으나 값을 하나씩 꺼낼때마다 count를 세서 정확하게 깊이를 파악하기 어려웠다.

이를 해결하기 위해 deque의 값에 depth를 추가해서 입력했다.
(1, 0)의 형태로 초기 값을 잡았고, depth가 2보다 작은 경우에만 측정하도록 하여 상근이의 친구 & 친구의 친구만 찾도록 설정하였다.
- depth의 경우 상근이의 친구를 찾는게 0, 친구의 친구를 찾는게 1이기 때문에 2보다 작은 값만 찾는것이 옳다.
