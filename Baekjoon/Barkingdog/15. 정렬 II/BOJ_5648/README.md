# 백준 5648번 - 역원소 정렬

[문제 보러 가기](https://www.acmicpc.net/problem/5648)

## 요약

모든 원소가 양의 정수인 집합이 있을 때, 원소를 거꾸로 뒤집고 그 원소를 오름차순으로 정렬하는 프로그램을 작성하세요.

단, 원소를 뒤집었을 때 0이 앞에 선행되는 경우는 0을 생략해야합니다.

## 입력

- 첫 번째로 입력되는 건 n (1 ≤ n ≤ 106)으로 사용자가 뒤이어 입력할 원소값을 결정합니다. 
- 입력하는 줄에는 하나의 원소값 뿐만 아니라 여러 원소값도 들어갈 수 있습니다.
- 단, 입력하는 정수는 1012을 넘어선 안 됩니다.

## 출력

- N개의 수를 거꾸로 뒤집고 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

## 예제 입출력

입력 및 출력 예시는 [문제 링크](https://www.acmicpc.net/problem/5648) 참고.

### 문제풀이

`1. 설계`

N개의 수를 입력받아서, 리스트에 저장한다.

처음 입력할 때, 하나의 원소값 뿐만 아니라 여러 원소값도 들어갈 수 있으므로 len을 활용해서 리스트에 모든 값이 저장되었는지 확인해야 한다.

역 원소로 뒤집어서 정렬을 해야 하므로, 먼저 str의 형태로 값을 받은 뒤에 뒤집어서 int형으로 저장하고 정렬한다.
