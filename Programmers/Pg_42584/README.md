# 42584 주식 가격

## 문제 설명

- 초 단위로 기록된 주식 가격이 담긴 배열 prices가 매개변수로 주어질 때, 가격이 떨어지지 않은 기간은 몇 초인지를 return

## 제한 조건

- prices의 각 가격은 1 이상 10,000 이하인 자연수
- prices의 길이는 2 이상 100,000 이하

## 입출력 예시

1. prices: `[1, 2, 3, 2, 3]`, return : `[4, 3, 1, 1, 0]`

- 1초 시점의 ₩1은 끝까지 가격이 떨어지지 않음
- 2초 시점의 ₩2는 끝까지 가격이 떨어지지 않음
- 3초 시점의 ₩3은 1초뒤에 가격이 떨어짐
- 4초 시점의 ₩2는 끝까지 가격이 떨어지지 않음
- 5초 시점의 ₩3은 끝까지 가격이 떨어지지 않음

### 문제 풀이

1. 설계

- 각 초별 변수의 크기를 가지고 값을 비교해야 한다.
- return을 미리 prices와 동일한 개수를 가지는 리스트로 미리 설정하고 비교
- prices[i]가 prices[k]보다 작아지는 순간 시간을 더이상 건드리지 않음

2. 수정점

- 처음에 check변수를 따로 두었는데, break를 사용하면 문제 없음
- check변수를 두더라도, 이중 for문을 활용하게 되는것은 똑같으므로, 주식 가격이 떨어지면 break를 하고 다음 prices의 answer를 측정하도록 변경
- break를 반드시 사용하여, 필요없이 모든 값을 확인하여 시간 복잡도가 $O(n^2)$이 되지 않도록 구성
