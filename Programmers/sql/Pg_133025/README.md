## 1. JOIN
    - 별칭 설정 가능
    - JOIN으로 두 테이블을 합치고, ON을 활용해서 조건을 설정 가능
    - 이 경우, 특정 테이블의 요소를 WHERE에서 선택이 가능
## 2.SUB_QUREY(IN)
    - IN의 경우, FLAVOR을 먼저 과일 베이스만 선택하는 것
    - IN ( )의 내부에 새로운 sql을 넣어서 설정
## 3.EXISTS
    - EXISTS를 설정하여 값이 있는지 없는지 판단
    - SELECT 1 -> 값이 있는것이 TRUE인지 확인
