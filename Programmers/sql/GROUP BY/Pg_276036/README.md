# 276036_언어별 개발자 분류하기

- GROUP BY를 활용하는 것이 깔끔.
  - SUM을 활용하면, 여러개의 행 중에서 원하는 값이 있다고 판단되면 그것을 더한다.
  - 즉, SUM(CATEGORY = 'Front End') > 0 이면, Front End의 능력이 있다고 판단하는 것과 같다.
    - case에서 when은 기존 프로그래밍 if절 처럼 위에서 부터 순서대로 체크한다.

- ID, EMAIL은 동일인이면, 모두 같은 값을 가지게 되므로 둘다 묶어서 하나의 그룹으로 판단해야 한다.
  - ID 또는 EMAIL만 묶게 된다면, 여러개의 행 중에서 어떤걸 보여줘야 할지 몰라 에러가 발생한다.
