# 131118, 서울에 위치한 식당 목록 출력하기

- 일단 리뷰 평균 점수를 조회해야 하고, 소수점 세 번째 자리에서 반올림
    - ROUND와 AVG를 활용
- 리뷰 평균 점수는 REVIEW 테이블에서, 나머지 정보는 INFO 테이블에서 찾을 수 있으므로, 두 테이블을 합쳐야 함.
  - JOIN을 사용하고, ON으로 동일한 조건인 REST_ID를 기준으로 합침
- 이때, 하나의 가게에 여러개의 리뷰가 있을 수 있고, 가게의 평균점수를 내려면 기준이 있어야 하므로 GROUP BY를 수행해야 함
  - GROUP BY를 하지 않은 경우 동일한 가게에 대해서 리뷰의 수 많큼 여러개의 줄이 생김
  - GROUP BY를 수행하면?
    - 가장 처음에 발견한 데이터 1개만을 보여주기 때문에, 1개의 줄만 남게 된다.
      - 이 상황에서 ROUND와 AVG를 활용하면 평균 점수를 구할 수 있다.
