# 12949 행렬의 곱셈

## 문제 설명

- 2차원 행렬 arr1, arr2를 입력받아, arr1에 arr2를 곱한 결과를 반환하는 함수, solution을 완성

## 제한 조건
- 행렬 arr1, arr2의 행과 열의 길이는 2 이상 100이하
- 행렬 arr1, arr2의 원소는 -10 이상 20 이하인 자연수
- 곱할 수 있는 배열만 주어진다.

## 입출력 예시
1. arr1: [[1, 4], [3, 2], [4, 1]]  
arr2: [[3, 3], [3, 3]]	  
return: [[15, 15], [15, 15], [15, 15]]  

- arr1에 arr2를 곱한 결과
```Text
arr1 * arr2
1 4    3 3
3 2 *  3 3
4 1
```
- arr1의 행의 수 3, arr2의 열의 수가 2이므로 결과는 3*2크기의 행렬이 나온다.
- arr1의 행과 arr2의 열을 곱한 결과가 각 자리의 값이 된다.
$$AB_{ij} = \sum_ka_{ik}b_{kj}$$ 
```Text
Return
15 15
15 15
15 15
```

2. arr1: [[2, 3, 2], [4, 2, 4], [3, 1, 4]]	  
arr2: [[5, 4, 3], [2, 4, 1], [3, 1, 1]]	  
return: [[22, 22, 11], [36, 28, 18], [29, 20, 14]]  
```Text
arr1 * arr2
2 3 2   5 4 3
4 2 4 * 2 4 1
3 1 4   3 1 1
```
```Text
Return
22 22 11
36 28 18
29 20 14
```
### 문제 풀이

1. 쉬운 방식
- numpy의 행렬관련 연산을 사용하면 매우 간단하다.
```Python
import numpy as np

def solution(arr1, arr2):
    arr1 = np.array(arr1)
    arr2 = np.array(arr2)
    answer = (arr1 @ arr2).tolist()
    return answer
```

2. 어려운 방식
- 직접 풀이를 구현해보자.
- 먼저 첫번째 배열의 행과 두번째 배열의 열을 가져와야 한다.
  - 첫번째 배열의 행을 가져오려면?
    - 간단하게 arr1[i][k]로 호출할 수 있다.
  - 두번째 배열의 열을 가져오려면?
    - arr2[k][j]로 호출할 수 있다.
- k는 어떻게 구하는가?
  - k는 `len(arr1[i])`과 같다.
- 그렇다면, k를 설정하고 $AB_{ij}$를 하나씩 구해서 answer를 생성하면 된다.
- answer는 2차원 배열이 되어야 하므로 미리 answer을 0을 가진 정답 크기의 2차원 배열로 설정하고 답을 갈아끼우면 된다.

3. 시간을 더 줄일 수 있는 방법은?
- Numpy 라이브러리는 C처럼 빠른시간에 계산이 가능하도록 구성된 코드라서 훨씬 빠르다
- 직접 구현한 파이썬 코드는 느릴수밖에없음..
