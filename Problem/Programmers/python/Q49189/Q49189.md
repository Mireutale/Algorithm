# 49189 가장 먼 노드

## 문제 설명
- n개의 노드가 있는 그래프에, 1번 노드에서 가장 멀리 떨어진 노드의 갯수를 구하려고 한다.
- 가장 멀리 떨어진 노드는 최단경로로 이동했을 때, 간선의 개수가 가장 많은 노드들을 의미함.

- 노드의 개수 n, 간선에 대한 정보가 담긴 2차원 배열 vertex가 매개변수로 주어질 때, 1번 노드로부터 가장 멀리 떨어진 노드가 몇 개인지를 return

## 제한 조건

- 노드의 개수 n은 2 이상 20,000 이하입니다.
- 간선은 양방향이며 총 1개 이상 50,000개 이하의 간선이 있습니다.
- vertex 배열 각 행 [a, b]는 a번 노드와 b번 노드 사이에 간선이 있다는 의미입니다.


## 입출력 예시
1. n = 3, vertex = [[3, 6], [4, 3], [3, 2], [1, 3], [1, 2], [2, 4], [5, 2]], result = 3

### 문제 풀이

1. 설계

- graph에 주어진 vertex의 값을 확인해서, 각 노드마다 인접한 노드들을 선택
- distance 배열을 만들고 1번 노드의 distance는 0으로 설정

- deque에 1번 노드와 인접해 있는 값들을 넣고 while문을 돌림.
- 다음 노드의 경우 현재 노드의 distance + 1로 처리
- 모든 값을 다 돌린 후, 거리 max인 노드를 count
