# 1843 - 사칙연산

- [문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/1843)

## 문제 설명

뺄셈은 덧셈과 달리 결합법칙이 적용되지 않는다.  
따라서, 연산 순서에 따라 그 결과가 바뀔 수 있다.  
문자열 형태의 숫자가 더하기 기호와 뺄셈 기호가 들어있는 배열 arr이 매개변수로 주어질 때 연산순서의 차이에 따라 계산 결과중 최댓값을 return

## 제한 조건
- arr는 두 연산자 "+", "-" 와 숫자가 들어있는 배열이며, 길이는 3 이상 201 이하 입니다.
  - arr의 길이는 항상 홀수입니다.
  - arr에 들어있는 숫자의 개수는 2개 이상 101개 이하이며, 연산자의 개수는 (숫자의 개수) -1 입니다.
  - 숫자는 1 이상 1,000 이하의 자연수가 문자열 형태로 들어있습니다.. (ex : "456")
- 배열의 첫 번째 원소와 마지막 원소는 반드시 숫자이며, 숫자와 연산자가 항상 번갈아가며 들어있습니다.

## 입출력 예시
1. 
- arr	= ["1", "-", "3", "+", "5", "-", "8"]	 
- result =  1

2. 
- arr = ["5", "-", "3", "+", "1", "+", "2", "-", "4"]	
- result = 3

### 문제 풀이
`1. 설계`

배열의 길이가 2n+1 이라고 하면 숫자는 n+1개가 있고, 연산자는 n개가 존재한다.

nums = [int(x) for x in arr[0::2]]로 숫자를 빼고, 
ops = [int(x) for x in arr[1::2]]로 연산자만 따로 선택

n = len(nums) 이다.

max_dp와 min_dp를 설정한다.
max_dp = [[float('-inf')] for _ in range(n)]
min_dp = [[float('inf')] for _ in range(n)]
max_dp는 최댓값을 가지도록, 모든 값을 -inf로 설정하고 min_dp는 최솟값을 가지도록 모든 값을 inf로 설정한다.

max_dp[i][j]는 i번째 숫자부터, j번째 숫자까지의 사이에서 최댓값을 가지는 경우를 의미한다.
min_dp[i][j]는 i번째 숫자부터, j번째 숫자까지의 사이에서 최솟값을 가지는 경우를 의미한다.

i == j 인 경우는 그냥 nums[i]를 가지도록 초기화 한다.

이제 구간을 설정한다.
d는 구간의 길이이다.
i는 시작점, j는 끝점이고, j = i + d가 성립한다.

1연산자는 숫자 1과2사이
2연산자는 숫자 2와3사이 ...

n연산자는 숫자 n과 n+1사이에 있다.
따라서, k는 i와 i+d사이에 있는 연산자 중에서 1개를 선택한다.

이 연산자가 +이냐, -이냐에 따라서, i-k구간에 있는 값과, k-j구간에 있는 값을 가져와서 저장한다.

연산자가 +이냐, -이냐에 따라서 더 작은 크기의 구간값을 구하는데 연산 순서를 다르게 지정해야 하기 때문.

결과로 max_dp[0][n-1]을 지정하면 1부터 n까지 전체 범위에서 최댓값을 출력하게 된다.
최솟값을 구하는 경우라면 min_dp[0][n-1]을 하면 된다.
