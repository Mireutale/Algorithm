# LeetCode 9번 - Palindrome Number

[문제 보러 가기](https://leetcode.com/problems/palindrome-number/description/)

## 문제 요약
정수 `x`가 주어질 때, 만약 정수 x가 palindrome이라면 true을 출력하고, 아니라면 false를 출력한다

palindrome = 뒤집었을 때 동일한 경우

## 예시

1. 
```text
Input: x = 121
Output: true
Explanation: 121 reads as 121 from left to right and from right to left.
```

2. 
```text
Input: x = -121
Output: false
Explanation: From left to right, it reads -121. From right to left, it becomes 121-. Therefore it is not a palindrome.
```

3. 
```text
Input: x = 10
Output: false
Explanation: Reads 01 from right to left. Therefore it is not a palindrome.
```

## 제한
- $-2^{31}$ <= $x$ <= $2^{31} - 1$
