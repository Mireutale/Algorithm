# LeetCode 3898번 - Find the Degree of Each Vertex

[문제 보러 가기](https://leetcode.com/problems/find-the-degree-of-each-vertex/)

## 문제 요약
$n \times n$ 크기의 2차원 정수 배열 `matrix`가 주어진다.
이 배열은 정점 `0`부터 `n - 1`까지 있는 무방향 그래프의 인접 행렬을 나타낸다.

- `matrix[i][j] == 1`: 정점 `i`와 정점 `j` 사이에 간선이 있다.
- `matrix[i][j] == 0`: 정점 `i`와 정점 `j` 사이에 간선이 없다.

정점의 차수(degree)는 해당 정점에 연결된 간선의 개수이다.
각 정점의 차수를 담은 길이 `n`의 정수 배열 `ans`를 반환한다.

## 예시

1. 
```text
Input: matrix = [[0,1,1],[1,0,1],[1,1,0]]
Output: [2,2,2]
Explanation:
Vertex 0 is connected to vertices 1 and 2, so its degree is 2.
Vertex 1 is connected to vertices 0 and 2, so its degree is 2.
Vertex 2 is connected to vertices 0 and 1, so its degree is 2.
```

2. 
```text
Input: matrix = [[0,1,0],[1,0,0],[0,0,0]]
Output: [1,1,0]
Explanation:
Vertex 0 is connected to vertex 1, so its degree is 1.
Vertex 1 is connected to vertex 0, so its degree is 1.
Vertex 2 is not connected to any vertex, so its degree is 0.
```

3. 
```text
Input: matrix = [[0]]
Output: [0]
Explanation:
There is only one vertex and it has no edges connected to it.
```

## 제한
- 1 <= n == matrix.length == matrix[i].length <= 100
- matrix[i][i] == 0
- matrix[i][j] is either 0 or 1
- matrix[i][j] == matrix[j][i]
