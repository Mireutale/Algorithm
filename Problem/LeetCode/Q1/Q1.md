# LeetCode 1번 - Two Sum

[문제 보러 가기](https://leetcode.com/problems/two-sum/description/)

## 문제 요약
Integers 배열로 이루어진 `nums`와 Integer `target`이 주어진다.
`nums`에서 두개의 서로 다른 수를 선택한 뒤 더해서 `target`을 만드는 정답을 찾는다.
정답은 배열의 위치를 출력하며 반드시 1개의 답만 존재하고, 동일한 수는 두번 나오지 않는다고 가정한다.

## 예시

1. 
```text
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].
```
2. 
```text
Input: nums = [3,2,4], target = 6
Output: [1,2]
```
3. 
```text
Input: nums = [3,3], target = 6
Output: [0,1]
```

## 제한
- 2 <= nums.length <= $10^4$
- $-10^9$ <= nums[i] <= $10^9$
- $-10^9$ <= target <= $10^9$
- Only one valid answer exists.

## Follow-up
- Can you come up with an algorithm that is less than O(n2) time complexity?
