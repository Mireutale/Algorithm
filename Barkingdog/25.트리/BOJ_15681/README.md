# 백준 15681번 - 트리와 쿼리

[문제 보러 가기](https://www.acmicpc.net/problem/15681)

## 문제 요약
가중치와 방향이 없는 임의의 루트 있는 트리가 주어진 경우, 아래 쿼리에 답을 내려라

- 정점 U를 루트로 하는 서브트리에 속한 정점의 수를 출력

## 입력
- 트리의 정점의 수 N과 루트의 번호 R, 쿼리의 수 Q가 주어진다. (2 ≤ N ≤ 105, 1 ≤ R ≤ N, 1 ≤ Q ≤ 105)
- 이어 N-1줄에 걸쳐, U V의 형태로 트리에 속한 간선의 정보가 주어진다. (1 ≤ U, V ≤ N, U ≠ V)
- 이는 U와 V를 양 끝점으로 하는 간선이 트리에 속함을 의미한다.
- 이어 Q줄에 걸쳐, 문제에 설명한 U가 하나씩 주어진다. (1 ≤ U ≤ N)
- 입력으로 주어지는 트리는 항상 올바른 트리임이 보장된다.

## 출력
- Q줄에 걸쳐 각 쿼리의 답을 정수 하나로 출력한다.

## 예제 입력 및 출력
링크의 예제 입력 및 출력 확인

## 문제풀이
```
9 5 3
1 3
4 3
5 4
5 6
6 7
2 3
9 6
6 8
5
4
8
```

정점의 수 9, 루트의 번호 5, 쿼리의 수 3
트리에 속한 간선의 정보 (1, 3), (4, 3), (5, 4), (5, 6), (6, 7), (2, 3), (9, 6), (6, 8)
문제에 설명한 U 5, 4, 8

서브 트리에 속한 정점의 수 이므로, U를 부모로 잡고, 그와 연결된 아래를 전부 서브 트리로 설정, 자식이 없는 노드만 선택하면 정점의 수를 구할 수 있다.

`2.재귀 사용에 따른 제한값 설정`
- count_subtree_nodes에 재귀 용법을 사용했으므로 제한값을 늘려줘야 한다.
  - sys.setrecursionlimit()로 제한값을 늘려준다.
