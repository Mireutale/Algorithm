# 백준 2660번 - 회장뽑기

[문제 보러 가기](https://www.acmicpc.net/problem/2660)

## 문제 요약
회원의 점수는 depth와 같다.
모두와 친구이면 1, 모두와 친구이거나 친구의 친구이면 2, ...
의 형태로 점수가 정해진다.

회장은 회원들 중 점수가 가장 작은 사람이 되고, 회장의 점수와 회장이 될 수 있는 모든 사람을 찾아라.

## 입력
- 입력의 첫째 줄에는 회원의 수가 있다. 
- 단, 회원의 수는 50명을 넘지 않는다. 
- 둘째 줄 이후로는 한 줄에 두 개의 회원번호가 있는데, 이것은 두 회원이 서로 친구임을 나타낸다. 
- 회원번호는 1부터 회원의 수만큼 붙어 있다. 
- 마지막 줄에는 -1이 두 개 들어있다.


## 출력
- 첫째 줄에는 회장 후보의 점수와 후보의 수를 출력하고, 두 번째 줄에는 회장 후보를 오름차순으로 모두 출력한다.

## 예제 입력 및 출력
링크의 예제 입력 및 출력 확인

## 문제풀이

`1.설계`

입력
```
5
1 2
2 3
3 4
4 5
2 4
5 3
-1 -1
```
출력
```
2 3
2 3 4 
```
회장의 점수는 2, 가능한 사람은 셋, 오름차순으로 2,3,4번이다.

친구의 경로를 보면
(1, 2), (2, 3), (3, 4), (4, 5), (2, 4), (5, 3)순이다.
1번의 친구는 2번 -> 2번의 친구인 3, 4번
2번의 친구는 1, 3, 4번 -> 3번의 친구인 5번
3번의 친구는 2, 4, 5번 -> 2번의 친구인 1번
4번의 친구는 2, 3, 5번 -> 2번의 친구인 1번
5번의 친구는 3, 4번 -> 3번의 친구인 2번

모두 가능한 경우는 2, 3, 4 뿐

점수를 cnt로 세고, friends를 개별로 만들어서 루프로 더하면 될 것 같다.
